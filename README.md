# Signal Runner

Signal Runner is a single-screen courier drone game. Your goal is to guide the drone from the start zone to the active delivery dock.

## Play

Open `index.html` in a browser or use the local server URL provided for the current build.

1. Select **Start Run**.
2. Move the courier drone with the arrow keys or `W`, `A`, `S`, `D`.
3. Navigate around the city walls and boundary barriers.
4. Reach **Dock 01**, the yellow delivery target, to complete the run.

## Current Build

Milestone 1 includes:

- Title/start screen
- Canvas playfield
- Keyboard movement
- Wall and boundary collision
- One visible delivery target
- Win screen after reaching the target

There is no timer, score, packet pickup, patrol enemy, gate, switch, pause, or level progression in this milestone.
