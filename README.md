# Signal Runner

Signal Runner is a 3D courier drone game. Your goal is to guide the drone through the city-grid playfield and dock at the active delivery relay.

## Play

Open `index.html` in a browser or use the local server URL for the current build.

1. Select **Start Run**.
2. Move the courier drone on the ground plane with the arrow keys or `W`, `A`, `S`, `D`.
3. Navigate around the visible 3D wall blocks and boundary barriers.
4. Reach **Dock 01**, the yellow delivery target, to complete the run.

## Current Build

Milestone 2 includes:

- 3D title/start overlay
- Three.js 3D playfield
- Courier drone player
- Fixed 3D camera view
- Keyboard movement
- Boundary collision
- 3D wall/block collision
- One visible and reachable Dock 01 delivery target
- Delivered/win screen after reaching Dock 01

There is no timer, packet pickup, score, patrol enemy, gate, switch, pause, level progression, or other later-milestone feature in this build.
