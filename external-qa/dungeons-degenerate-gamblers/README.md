# Dungeons & Degenerate Gamblers Demo Player Manual

Primary source URL: `https://purplemosscollectors.itch.io/dndg`

Direct official browser build URL for retrying provider-wrapper or embedded-viewport issues:
`https://html-classic.itch.zone/html/15020933/index.html`

## Objective

Play the browser demo as a card-based roguelike encounter. Use the visible tutorial and card text to choose legal actions, resolve card effects, and progress as far as the demo allows.

## Controls

- Start the game from the visible player controls. On the primary itch.io page, use the embedded player. On the direct official browser build, use the game menu directly.
- Use mouse or touch input to select cards, buttons, rewards, and menu options.
- Read visible card text, tooltips, prompts, and tutorial messages before choosing actions.
- Use visible pause, settings, restart, give-up, or volume controls only if the game exposes them.

## Player-Facing Behaviors To Expect

- Card choices should show readable names, costs or values, and consequences.
- Legal and illegal actions should be understandable from visible state.
- Settings, if present, should include visible audio or volume controls.
- The game may be a demo; a complete final ending may not be available.
