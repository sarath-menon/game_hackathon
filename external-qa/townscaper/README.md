# Townscaper Web Player Manual

Source URL: `https://oskarstalberg.com/Townscaper/`

## Objective

Use the browser version as a creative building toy. Place and remove blocks to create structures and inspect how the scene responds.

## Controls

- Use the visible controls shown by the game if available.
- Common controls are expected to include mouse clicks for placing blocks, alternate click or tool selection for removing blocks, and mouse drag or wheel controls for camera movement.
- Use fullscreen or browser zoom only if the WebGL view is hard to interact with.

## Player-Facing Behaviors To Expect

- The scene should load into an interactive WebGL view on supported desktop browsers.
- Clicks should place or modify blocks with immediate visual feedback.
- Camera movement should allow inspection of the structure.
- The browser version may have a smaller build grid and may not support saving.
- Mobile browsers may be unsupported.
