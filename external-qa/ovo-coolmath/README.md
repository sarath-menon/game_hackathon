# OvO Player Manual - Coolmath Retry

Source URL: `https://www.coolmathgames.com/0-ovo`

## Objective

Complete platforming levels by moving through the level, avoiding hazards, collecting optional coins if reachable, and reaching the flag as quickly and cleanly as possible.

## Controls

- Use arrow keys to move and jump.
- Press `Down` while running to slide.
- Jump while sliding to go farther in the air.
- Press `Down` while in the air to smash through transparent platforms.
- Jump next to a wall while sliding down it to wall jump.
- Click inside the game frame first if keyboard input does not work.

## Player-Facing Behaviors To Expect

- The game should reach an interactive menu or playable level from the Coolmath page.
- Movement, jumping, sliding, smashing, hazards, level reset, and completion should be visually clear.
- Reaching the flag should complete or advance the level.
- Settings, options, skin, mode, or audio controls should be reachable from visible UI if present.
