# OvO Classic Player Manual - Poki Retry

Source URL: `https://poki.com/en/g/ovo-classic`

## Objective

Complete speedrun platforming levels by moving through the level, avoiding hazards, and reaching the finish flag as quickly and cleanly as possible.

## Controls

- Use left and right arrow keys to move.
- Press `Space` to jump.
- Press `Down` to slide.
- Use controls displayed inside the game if they differ from the provider page.
- Click inside the game frame first if keyboard input does not work.

## Player-Facing Behaviors To Expect

- The game should reach an interactive menu or playable level from the Poki page.
- Movement, jumping, sliding, wall-jump or advanced movement prompts, hazards, level reset, and completion should be visually clear.
- Reaching the finish flag should complete or advance the level.
- Settings, options, audio, or fullscreen controls should be reachable from visible UI if present.
