# PolyTrack Player Manual - Itch.io Retry

Source URL: `https://kodub.itch.io/polytrack`

## Objective

Drive the time-trial track as cleanly as possible. Stay on the visible road, follow the track route, and use restart if the car becomes stuck or the run is no longer useful.

## Controls

- Use `WASD` or arrow keys to drive.
- Use `R` or `Enter` to restart the track.
- Click inside the game frame first if keyboard input does not work.
- Use fullscreen only if the page offers it and keyboard focus is unreliable.

## Player-Facing Behaviors To Expect

- The game should reach a playable menu or track from the Itch.io page.
- The track should be readable from the starting camera.
- Steering and throttle should move the car after the game has keyboard focus.
- The game should provide a visible route, car position, and timing/run feedback.
- Restart should return the run to a clean starting state.
- Any graphics, audio, control, editor, or quality options should be reachable from visible UI if present.
