# OvO Player Manual

Source URL: `https://www.crazygames.com/game/ovo`

## Objective

Complete platforming levels by moving through the level, avoiding hazards, and reaching the goal as quickly and cleanly as possible.

## Controls

- Use the controls shown by the game or its start/options UI.
- Common controls are expected to include left/right movement, jump, slide or crouch, and restart.
- Click inside the game frame first if keyboard input does not work.
- If the game offers control remapping, difficulty, or mode options, use the visible options menu.

## Player-Facing Behaviors To Expect

- Movement should respond immediately after the game frame has focus.
- Jumping, sliding, momentum, hazards, level reset, and completion should be visually clear.
- Difficulty or mode choices should visibly change the selected mode if present.
- Key remapping or settings changes should be reflected by the options UI if present.
