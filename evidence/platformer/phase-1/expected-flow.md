# Expected Flow: Side-Scrolling Platformer Phase 1

The recording should show the hosted Skyline Stepper game loading at the title/start screen. The player starts the run with Enter, briefly moves left, then moves right through the side-scrolling rooftop route.

During the run, the character should jump with W across solid platforms while gravity returns the character to platform or ground surfaces. The camera should follow the character as the level scrolls, keeping the route readable and the exit objective understandable.

The run should end when the character reaches the green EXIT beacon. The game should stop normal play and show the Level Complete state.
