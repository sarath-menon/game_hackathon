# Expected Flow: Side-Scrolling Platformer Phase 3

The recording should show Skyline Stepper loading at the title/start screen, then starting the run. The HUD should show health, checkpoint state, glow-core count, and distance to the exit.

The run should use documented controls to move through the readable lower-deck route, collect at least one glow core, pass the mid-level checkpoint, preserve side-scrolling camera behavior, reach the green EXIT beacon, and show `Level Complete` with collected cores and remaining health.

The recording should also demonstrate restart from completion with Enter, resetting health, checkpoint, core count, and distance. Separate black-box probes should verify visible upper-route hazards/patrols, checkpoint respawn after damage, and health-zero failure summary.
